## Overview

Run multiple workspaces at once. Each is an isolated git worktree with its own branch, terminal, and ports.

![Multiple Workspaces](/images/branches.png)

## Use Cases

- **Parallel features** - Work on multiple features simultaneously
- **AI agents** - Run Claude Code in separate workspaces
- **Code review** - Review PRs while continuing your work

## Port Management

Each workspace gets dedicated ports to avoid conflicts. View active ports in the ports panel.
