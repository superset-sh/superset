## General

**What platforms?**
macOS only. Windows/Linux coming soon.

**How does it work?**
Uses git worktrees (or jj workspaces for Jujutsu repos) to give each branch its own directory.

**Do I need an account?**
Yes, for team features and settings sync.

## Workspaces

**Same branch in multiple workspaces?**
No. One branch = one workspace. Create a new branch if needed.

**Where are workspaces stored?**
In your Superset data directory as git worktrees or jj workspaces.

## Terminal

**Which shell?**
Your system default (bash/zsh) with your config files.

**Do sessions persist?**
Yes, terminals survive app restarts.

## Git & Jujutsu

**Does it modify git config?**
No. Uses standard git/jj commands.

**Does it support Jujutsu (jj)?**
Yes. Superset auto-detects colocated jj repos and uses native `jj workspace` commands. The `jj` CLI must be installed (`brew install jj`). If not installed, Superset falls back to git.

**Private repos?**
Yes. Configure GitHub auth: `gh auth status`
