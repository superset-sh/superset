## System Requirements

- **macOS**: Apple Silicon (M1/M2/M3) or Intel
- **Git**: Must be installed and configured
- **GitHub CLI**: Required for GitHub authentication (`gh auth status`)

## Setup Steps

1. **Download** the app from the button above
2. **Move** Superset.app to your Applications folder
3. **Open** Superset and sign in with your account
4. **Verify** GitHub authentication by running `gh auth status` in terminal

### Build from Source

You can also build Superset from source on any platform:

```bash
git clone https://github.com/superset-sh/superset.git
cd superset
bun install
bun run build
```

See our [GitHub repository](https://github.com/superset-sh/superset) for more details.
