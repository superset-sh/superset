## Overview

Run AI coding agents in isolated workspaces. Each agent works independently without interfering with others.

![AI Agents](/images/agents.png)

## Supported Agents

- **Claude Code** - Anthropic's CLI
- **Codex** - OpenAI's assistant
- **OpenCode** - Open-source alternative
- **Gemini CLI** - Google's CLI agent
- **Cursor Agent** - Cursor AI agent

## Chat

Superset includes a built-in chat panel for AI interactions without leaving the app.

- **Model picker** - Choose from Claude (Anthropic), GPT-4o, o3, Codex (OpenAI), and others
- **Anthropic OAuth** - Connect your Anthropic account in the model picker for Claude models
- **OpenAI API key** - Enter your OpenAI API key in the model picker to access GPT-4o, o3, and Codex models
- **Auto-generated titles** - Chat session titles are generated automatically from the conversation

## Usage

Run agents directly in the terminal:

```bash
claude "Add error handling to fetchUser"
```

## Parallel Agents

Create multiple workspaces and run an agent in each. Compare results and merge the best solution.

## Inline Prompts

Agents can pause and ask for input directly in the chat pane:

- **Questions** - Answer via multiple choice or free text
- **Tool approvals** - Approve, decline, or always-allow a tool call
- **Plan reviews** - Review the agent's plan and approve or request changes with optional feedback

Respond inline without switching context.

## Sandbox Access

When an agent needs access to files outside its workspace, it will prompt for permission in the chat. Approve or deny the request to grant or restrict access.

## Notifications

Get notified when agents finish. Configure sounds in Settings.
