## Overview

Built-in terminal per workspace. Sessions persist across app restarts.

## Features

- **Multiple tabs** - Open several terminals per workspace
- **Persistence** - Sessions survive restarts
- **Clickable links** - URLs and file paths are clickable

## Shortcuts

| Action | Shortcut |
|--------|----------|
| New tab | ⌘T |
| Close tab | ⌘W |
| Clear | ⌘K |

## Environment Variables

Available in all terminal sessions:

| Variable | Description |
|----------|-------------|
| `SUPERSET_ROOT_PATH` | Path to root repository |
| `SUPERSET_WORKSPACE_NAME` | Current workspace name |
