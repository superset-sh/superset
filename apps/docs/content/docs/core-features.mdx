## Core Features

Superset has a few core features that are worth knowing about.

### Workspace Management

Superset allows you to manage your workspaces in a way that is easy to understand and use.

### Terminal Integration

Superset allows you to integrate your terminal into your workspace.

### Team Collaboration

Superset allows you to collaborate with your team.

### How It Works
