## Customize

Open Settings → Keyboard Shortcuts to customize any hotkey.

## Adaptive Layout Mapping

Enable **Adaptive layout mapping** in Settings → Keyboard Shortcuts to remap shortcuts to physical key positions on non-QWERTY layouts (e.g., ⌘Z on QWERTY maps to the Z-position key on QWERTZ).

- **On** - Shortcuts remap to equivalent physical positions across layouts
- **Off** (default) - Shortcuts dispatch exactly as bound, regardless of layout

## Import/Export

Share your hotkey configuration:
- **Export** - Save your shortcuts to a file
- **Import** - Load shortcuts from a file
