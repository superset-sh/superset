## Default Shortcuts

| Action | Shortcut |
|--------|----------|
| New workspace | ⌘N |
| Open in editor | ⌘O |
| New terminal tab | ⌘T |
| Close tab | ⌘W |
| Clear terminal | ⌘K |
| Settings | ⌘, |
