## Chat UI Overhaul <PRBadge url="https://github.com/superset-sh/superset/pull/1693" /> <PRBadge url="https://github.com/superset-sh/superset/pull/1697" />

The chat interface got a significant visual refresh. Streaming messages now animate with a progressive typing effect, tool output panels are collapsible to reduce noise, and loading/thinking states use a unified compact shimmer. The input composer has been redesigned with pill-shaped toolbar buttons, a circular submit button, and a cleaner layout with a hint row showing `@` mentions and `/commands` shortcuts.

- Progressive typing animation for streaming messages
- Collapsible tool output panels with inline status icons
- Pill-style controls for model picker, permission mode, and thinking toggle
- Permission mode labels clarified: Auto, Semi-auto, and Manual

## New Project Page <PRBadge url="https://github.com/superset-sh/superset/pull/1623" />

Project creation now has a dedicated full-page experience instead of a dialog. Navigate to it from the welcome screen or the sidebar footer and choose between three tabs: create an empty Git repo, clone from a URL, or (coming soon) start from a template. A directory picker defaults to `~/.superset/projects` with a native browse button.

## Multi-Agent Presets <PRBadge url="https://github.com/superset-sh/superset/pull/1630" /> <PRBadge url="https://github.com/superset-sh/superset/pull/1625" /> <PRBadge url="https://github.com/superset-sh/superset/pull/1620" />

Superset now ships built-in terminal presets and hook support for Copilot, OpenCode, and Gemini CLI alongside the existing Claude and Codex presets. Each agent gets the correct default flags (`--allow-all` for Copilot, `-y` for Gemini) and Superset notification hooks so task completions surface in the UI.

## Configurable Worktree Location <PRBadge url="https://github.com/superset-sh/superset/pull/1626" />

Set a custom base directory for new worktrees in Settings > Behavior. You can also override this per-project in Project settings. The per-project setting takes precedence, with a reset option that falls back to the global default.
