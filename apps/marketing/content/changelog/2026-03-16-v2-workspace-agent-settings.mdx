## V2 Workspace Sidebar & New Workspace Modal <PRBadge url="https://github.com/superset-sh/superset/pull/2399" /> <PRBadge url="https://github.com/superset-sh/superset/pull/2391" /> <PRBadge url="https://github.com/superset-sh/superset/pull/2506" />

The workspace sidebar has been redesigned from the ground up. Workspaces are now organized by project, and the new workspace modal has a prompt-first flow that lets you describe what you want to build before picking a branch or repository.

- Project-scoped workspace grouping in the sidebar
- Prompt-first creation flow — describe your task, then configure workspace details
- V2 project selection integrated directly into the creation flow

## Configurable Agent Settings <PRBadge url="https://github.com/superset-sh/superset/pull/2495" />

{/* TODO: Add screenshot of the agent settings panel */}

Agent behavior is now configurable per workspace. Adjust model, instructions, and other agent parameters directly from settings without editing config files manually.

## Merge Tabs via Drag-and-Drop <PRBadge url="https://github.com/superset-sh/superset/pull/2488" />

Drag any tab onto another to merge them into a single pane. This makes it easier to consolidate your layout without manually rearranging splits.

## Pane Header Rename <PRBadge url="https://github.com/superset-sh/superset/pull/2487" />

You can now rename pane headers directly in the UI, making it easier to label and organize complex multi-pane layouts.

## Linked Linear Issue Pills <PRBadge url="https://github.com/superset-sh/superset/pull/2383" />

Linear issues referenced in agent tasks now render as inline pills with the issue title and status. Mention chips let you quickly reference tasks in chat without leaving context.

## Improvements

- **Keyboard shortcut to equalize pane splits** - Press `Cmd+Shift+0` to instantly balance all pane splits to equal widths <PRBadge url="https://github.com/superset-sh/superset/pull/2381" />
