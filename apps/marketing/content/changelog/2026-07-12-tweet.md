What we shipped this week @ Superset 🛳️

1. Rich input for terminal agents ⌨️

CLI agents are where the work happens, but a raw TTY line is a terrible place to write a real prompt: no multiline editing, no mentions, and pasting can execute line-by-line.

So we brought the workspace chat composer into the terminal. Press ⌘I over Claude Code, Codex, or any CLI agent and you get a real editor — Shift+Enter newlines, @file mentions, drafts that survive tab switches — and your prompt lands in the TUI as one clean block. The terminal stays the terminal; the input just stops fighting you.

2. v2 is now the default 🚀

Every new signup now lands directly in the v2 experience — multi-pane workspaces, the activity strip, custom agents, all of it. Existing users keep their setup. This is the milestone we've been driving at all through open beta.

3. Live agent sessions on iOS (experimental) 📱

Start and drive a coding agent running on your desktop, from your phone. The agent, your code, and permissions never leave your machine — the iOS app is a live remote UI over our relay. Answer permission prompts, review plans and tool calls, switch models from the couch. Enable it in Desktop → Settings → Security.

4. Mistral Vibe 🤝

Mistral's Vibe CLI is now a first-class Superset agent — its own icon in every picker, a model selector, working indicator, completion chime. We also added GPT-5.6 Sol, Terra, and Luna to the Codex picker. Bring whatever agent you like; Superset is where they all run.

Also this week: wake offline hosts with a configurable wake command, remote hosts on every plan (relay no longer needs a paid subscription), a Sydney relay region, Claude Code transcript scrolling at full speed, and Linear project/cycle filters for Tasks.

As always, tons of bug fixes and improvements, thanks to all our users in open beta for V2 helping us make the product better everyday <3
