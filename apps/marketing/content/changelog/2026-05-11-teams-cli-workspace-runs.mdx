## Teams <PRBadge url="https://github.com/superset-sh/superset/pull/4403" />

Teams are now a first-class org primitive. Create teams to group members, assign workspaces and automations to a team, and filter the automations view between **Mine** and **Team** runs.

## CLI: `projects create`, `projects setup`, and `workspaces open` <PRBadge url="https://github.com/superset-sh/superset/pull/4355" /> <PRBadge url="https://github.com/superset-sh/superset/pull/4356" /> <PRBadge url="https://github.com/superset-sh/superset/pull/4258" />

Three new commands extend the `superset` CLI:

- `superset projects create` — scaffold a new project from your shell
- `superset projects setup` — link an existing local repo to a Superset project
- `superset workspaces open` — open a workspace directly from the terminal

## Preset-backed workspace runs <PRBadge url="https://github.com/superset-sh/superset/pull/4335" />

Workspaces can now be started with a preset, so the agent launches with the right configuration without any manual setup. Useful for automation flows and repeatable tasks.

## Copy actions in diff comment threads <PRBadge url="https://github.com/superset-sh/superset/pull/4389" />

Diff comment threads now have a copy action so you can grab a comment's permalink or content without leaving the review view. Thread layout is also tighter to keep more context on screen.
