## Mobile-First Agents Experience <PRBadge url="https://github.com/superset-sh/superset/pull/2595" />

The web app now has a fully redesigned agents experience optimized for mobile. You can browse, start, and monitor agents from your phone with an interface built for smaller screens.

## Configurable Light/Dark Themes <PRBadge url="https://github.com/superset-sh/superset/pull/2557" />

{/* TODO: Screenshot of the theme picker / system mode toggle */}

When your system is set to auto mode, you can now independently configure which Superset theme is used for light and which is used for dark. Mix and match any of the available themes rather than being locked to the default pair.

## Open in Editor from Worktrees <PRBadge url="https://github.com/superset-sh/superset/pull/2999" />

Worktrees now have an "Open in Editor" action so you can jump straight from a worktree into your external editor without navigating away.

## Cmd-Click File Paths Opens External Editor <PRBadge url="https://github.com/superset-sh/superset/pull/2903" />

File paths mentioned in chat or output can now be Cmd-clicked to open directly in your configured external editor.

## Improvements

- **New workspace modal UX** - The new workspace modal replaces the plus dropdown with icon buttons for a cleaner layout, and PR search now runs server-side for faster, more complete results <PRBadge url="https://github.com/superset-sh/superset/pull/2980" /> <PRBadge url="https://github.com/superset-sh/superset/pull/2909" />
- **Close workspace hotkey** - Added `Cmd+W` hotkey and context menu option to close a workspace without reaching for the mouse <PRBadge url="https://github.com/superset-sh/superset/pull/2907" />
- **Settings search indicator** - Settings panel now shows a visual indicator when a search filter is active, so it's clear why some options are hidden <PRBadge url="https://github.com/superset-sh/superset/pull/2997" />
- **Terminal flow control** - Hardened terminal host backlog and flow control to prevent output drops under heavy load <PRBadge url="https://github.com/superset-sh/superset/pull/3006" />
