## Terminal sessions survive app updates <PRBadge url="https://github.com/superset-sh/superset/pull/3971" />

The PTY daemon now hands off file descriptors when it upgrades, so your running terminal sessions stay alive through app updates and daemon binary replacements. No more losing your shell state when Superset updates in the background.

## TypeScript SDK <PRBadge url="https://github.com/superset-sh/superset/pull/3937" />

`@superset_sh/sdk` is a new TypeScript package that mirrors the CLI 1:1 — manage tasks, automations, workspaces, and more from your own scripts or apps without shelling out to the CLI.

## Automation prompt version history <PRBadge url="https://github.com/superset-sh/superset/pull/3996" />

Automation prompts now keep a version history. Browse previous versions and restore any of them — no more accidentally overwriting a prompt you wanted to keep.

## V2 changes tab: stage, unstage, and discard <PRBadge url="https://github.com/superset-sh/superset/pull/4022" />

The v2 Changes tab now has inline controls for staging, unstaging, and discarding individual files — no need to drop into the terminal for basic git operations.
