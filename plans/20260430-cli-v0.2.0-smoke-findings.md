## ✅ Worth keeping

These felt right and shouldn't drift:

- "Did you mean..." suggestion on unknown commands
- `workspaces create` errors with clear hints for missing/conflicting `--host`/`--local`
- `tasks get nonexistent` → "Error: Task not found: <slug>" + exit 1
- enum error: "Error: --priority: invalid value 'critical'" + "Hint: Valid values: urgent, high, medium, low, none"
- number error: "Error: --limit: expected a number, got 'abc'"
- `automations list` table with friendly schedule (`Daily at 9:00 AM`) and next run (`May 1, 2026 at 9:00 AM PDT`)
- `auth whoami` and `status` human output is concise (modulo the unit issues #30 #32)
- Exit codes are 1 for errors (after the unknown-subcommand bug is fixed)
